## INSTALL Gemfile
```
source 'https://rubygems.org'

gem 'jekyll'
gem 'redcarpet'
gem 'jekyll-gist'
gem 'jekyll-paginate'
gem 'pygments.rb'
```

## BUNDLER
```
bundler install
```

## SETTING UP ENVIRONMENT
```
bundler exec bash
```

## init
```
jekyll new .
```

## config
```
_config.yml
```

## preview
```
jekyll server
```

## preview dameon mode
```
jekyll serve --detcah
```

## build site

```
jekyll build
```

## build site and watch for changes

```
jekyll build --watch
```
